Composite Score

Used to sort the heatmap. Based on 4 core metrics: query size, response size, server time, throughput. Each is percentile-ranked (0 = best, 1 = worst). Missing metrics excluded from average. Coverage penalty: +0.1 per missing core metric if <3 available. Tier 3 values pushed 20% toward midpoint. Final = mean(adjusted ranks) + coverage penalty. Lower = better = left in heatmap.
